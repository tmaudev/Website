Astronomical Library Testing

Task

Propagate and compare heliocentric planetary ephemeris data from old and new versions of The Aerospace Corporation's astronomical library, AstroLib.

Accomplishments

Developed a benchmark program (in C) to test the validity of planetary information given various input dates.
Compared information with the USNO Astronomical Almanac to identify discrepancies in an unreleased update of AstroLib.

Impact

Error identification allowed AstroLib developers to update library to accurately represent planetary data.
Benchmark testing will verify correct data for future versions, ensuring valid planetary data for development and analysis of interplanetary trajectories.
