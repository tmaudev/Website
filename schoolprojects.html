California Polytechnic State University, San Luis Obispo

General Information


Current Year: Fourth
Major: Computer Engineering
Graduation: Spring 2016


After careful consideration of many universities, I chose to attend Cal Poly, San Luis Obispo. The "Learn By Doing" approach towards academics greatly appealed to my strength in kinesthetic learning.

Looking back as a fourth year Computer Engineering (CPE) major, I couldn't be happier with my choice of university. Cal Poly's CPE program continues to be an amazing learning experience full of challenging courses that fulfill my need/passion to constantly learn new programming concepts and languages. I can't wait to see what these upcoming quarters will have to offer!


Note: Below, you can find details/information on my completed/enrolled computer-related courses. Course descriptions have been taken from the Cal Poly catalog.

First Year (2011 - 2012)

Fundamentals of Computer Science for Scientists and Engineers

Course Number: CSC 235
Primary Language: C++

Description

Introduction to the fundamentals of computer programming with an emphasis on mathematical, scientific and engineering applications: principles of algorithmic problem solving and procedural programming using a modern programming language, data types, elementary data structures, input/output and control structures.

Second Year (2012 - 2013)

Fundamentals of Computer Science I

Course Number: CPE/CSC 101
Primary Language: C

Description

Basic principles of algorithmic problem solving and programming using methods of top-down design, stepwise refinement and procedural abstraction. Basic control structures, data types, and input/output. Introduction to the software development process: design, implementation, testing and documentation.


Fundamentals of Computer Science II

Course Number: CPE/CSC 102
Primary Language: Java

Description

Basic design, implementation, testing, and documentation of object-oriented software. Introduction to classes, interfaces, inheritance, algorithms (sort, search, recursion), abstract data types, data structures (lists, stacks, queues), file I/O, and exceptions.


Discrete Structures

Course Number: CSC 141
Primary Language: N/A

Description

Introduction to structures of computer science: logic, sets, relations, functions, graphs and trees. Propositional and predicate logic. Applications of predicate logic to preconditions, postconditions, and proof techniques. Introduction to complexity of algorithms.


Digital Design

Course Number: CPE 133
Primary Language: VHDL

Description

Number systems, Boolean algebra, Boolean functions, and function minimization. Analysis and design of combinational and sequential logic circuits. Hardware Description Language (HDL) concepts and applications digital design and synthesis in Programmable Logic Devices (PLDs).

Third Year (2013 - 2014)

Fundamentals of Computer Science III

Course Number: CPE/CSC 103
Primary Language: Java

Description

Introduction to data structures and analysis of algorithms. Abstract data types. Specification and implementation of advanced data structures. Theoretical and empirical analysis and proofs of properties of recursive and iterative algorithms. Software performance evaluation and testing techniques.


Systems Programming

Course Number: CPE/CSC 357
Primary Language: C

Description

C programming language from a system programming perspective. Standard C language including operators, I/O functions, and data types in the context of system functions. Unix commands, shell scripting, file system, editors.


Computer Design and Assembly Language Programming

Course Number: CPE 233
Primary Language: VHDL & RAT Assembly

Description

Design and implementation of digital computer circuits via CAD tools for programmable logic devices (PLDs). Basic computer design with its datapath components and control unit. Introduction to assembly language programming of an off-the-shelf RISC-based microcontroller.

Fourth Year (2012 - 2013)

Computer Architecture

Course Number: CPE/CSC 315
Primary Language: Java

Description

In-depth study of the instruction set architecture and hardware design of a specific CPU. Introduction to pipelines, input/output and multi-processors. Computer abstractions and performance measurement.


Operating Systems

Course Number: CPE/CSC 453
Primary Language: Java

Description

Introduction to sequential and multiprogramming operating systems; kernel calls, interrupt service mechanisms, scheduling, files and protection mechanisms, conventional machine attributes that apply to operating system implementation, virtual memory management, and I/O control systems.


Introduction to Computer Networks (Currently Enrolled)

Course Number: CPE/CSC 464
Primary Language: Unknown

Description

Computer network architectures; communications protocol standards; services provided by the network; historical and current examples presented.
